# 🚀 JINSI YA KUPELEKA TYRE PLAZA WEBSITE GITHUB PAGES

## HATUA 1 — Unda Account GitHub (kama huna)
1. Nenda: https://github.com
2. Bonyeza **Sign up**
3. Jaza jina la mtumiaji, barua pepe, nenosiri
4. Thibitisha barua pepe yako

---

## HATUA 2 — Unda Repository Mpya

1. Ingia GitHub → bonyeza **"+"** kona ya juu kulia
2. Chagua **"New repository"**
3. Jaza kama hivi:
   ```
   Repository name: tyreplaza
   Description:     Tyre Plaza – Matairi, Betri & Rims | Mwanza Tanzania
   Visibility:      ✅ Public
   ```
4. **USICHAGUA** "Initialize this repository" — acha tupu
5. Bonyeza **"Create repository"**

---

## HATUA 3 — Pakia Faili kwa GitHub Desktop (Rahisi Zaidi)

### A) Pakua GitHub Desktop
- Nenda: https://desktop.github.com
- Pakua na install

### B) Clone Repository
1. Fungua GitHub Desktop
2. File → Clone Repository
3. Weka URL: `https://github.com/JINA_LAKO/tyreplaza`
4. Chagua folder kwenye kompyuta yako
5. Bonyeza Clone

### C) Nakili Faili
- Nakili faili ZOTE kutoka kwa folder uliyopakua (tyreplaza/)
- Pakia kwenye folder ya repository uliyoclone

### D) Commit na Push
1. GitHub Desktop itaona mabadiliko
2. Summary: `Tyre Plaza Website v2.0`
3. Bonyeza **"Commit to main"**
4. Bonyeza **"Push origin"**

---

## HATUA 4 — Washa GitHub Pages

1. Nenda repository yako GitHub
2. Bonyeza **Settings** (kona ya juu)
3. Scroll chini → **Pages** (upande wa kushoto)
4. Chini ya **Source** → chagua:
   ```
   Branch: main
   Folder: / (root)
   ```
5. Bonyeza **Save**
6. Subiri dakika 2-3
7. URL yako itakuwa:
   ```
   https://JINA_LAKO.github.io/tyreplaza/
   ```

---

## HATUA 5 — Thibitisha Website Inafanya Kazi

Bonyeza URL — utaona website ya Tyre Plaza ikifunguka! ✅

---

## 🌐 DOMAIN YA KWELI (hiari)

Kama unataka `www.tyreplaza.co.tz`:

1. Nunua domain: Afriregister, TzDomains, au GoDaddy
2. Unda faili `CNAME` kwenye root ya repository:
   ```
   tyreplaza.co.tz
   ```
3. DNS settings za domain yako — ongeza records:
   ```
   A    @    185.199.108.153
   A    @    185.199.109.153
   A    @    185.199.110.153
   A    @    185.199.111.153
   CNAME www  JINA_LAKO.github.io
   ```
4. GitHub Pages Settings → Custom domain → weka `tyreplaza.co.tz`
5. Washa **Enforce HTTPS** ✅
